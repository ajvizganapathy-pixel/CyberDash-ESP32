Capabilities

Everything a network
needs to be seen.

CyberDash packs professional-grade network visibility into a $5 microcontroller. No Raspberry Pi. No laptop. Just the ESP32.

📡
WiFi Network Scanner
Continuously scans all 2.4 GHz channels. Logs SSIDs, BSSIDs, channel, RSSI, encryption type, and beacon interval. Detects hidden networks and tracks signal history.
🛡
Threat Alerts
Real-time heuristic detection of ARP spoofing, probe floods, rogue APs, and unexpected MAC changes.
WARN ARP reply mismatch · .27
INFO New device joined · .41
CRIT Probe flood · chan 6
WARN RSSI drop · .14
⌨
Serial Console
Full REPL accessible over USB or WebSerial. Query live state, tune thresholds, reload config.
cyber> scan --channels all Scanning 13 channels... [OK] Found 9 networks cyber>
📶
RSSI Spectrum Monitor
Per-channel signal strength visualisation. Instantly spot interference, crowded channels, and weak clients.
🔬
Packet Inspector
Promiscuous-mode capture. Filter by MAC, type, or channel. Raw frame hex dump with annotation.
🔒
Fully Offline
Zero external requests. The entire frontend is baked into flash. Runs as a local AP — no internet required, ever.
⬆
OTA Updates
Push new firmware from the dashboard itself. Diff-based incremental update with rollback protection.

Getting Started

Up and running
in 4 steps.

No SDKs to install beyond the Arduino IDE and the ESP32 board package. Clone, compile, flash.

01
Clone the repo
git clone https://github.com/
ajvizganapathy-pixel/
CyberDash-ESP32 Open in Arduino IDE or PlatformIO.
→
02
Install dependencies
Install ESPAsyncWebServer, AsyncTCP, and the LittleFS plugin from the Arduino Library Manager. Board: ESP32 by Espressif.
→
03
Upload SPIFFS data
Run the LittleFS upload tool to flash the data/ folder containing the web interface to the chip's filesystem.
→
04
Connect & explore
Join the CyberDash-AP WiFi and open 192.168.4.1 in any browser. The full dashboard loads instantly.
